1. Root is always black.
  2. Red nodes can't have red children.
  3. Every path from root to leaf has the same number of black nodes.

Adding a Node:

0-9 enter a value
Enter add the node
Escape cancel

Editing:

Right Click on 2 nodes to rotate
Right Click twice on a node to swap colors
For every node on path of insertion (in order from top to bottom): If node has two RED children then: color the children BLACK color the node RED (unless it's the root) Fix any (BLACK -> RED -> RED) paths If you have a straight 3 nodes (all left children or all right children) Rotate the top RED node with it's parent and swap their colors If you have a kink (one is left child, one is right child) Rotate the two REDs to form a straight then fix the straight
